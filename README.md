        <h2>A) ¿Qué es la reforestación?</h2>
        <p>Es el proceso de proteger árboles, monitorear su crecimiento y restaurar el ecosistema. También implica educar a la comunidad para que se involucre.</p>
    </section>

    <section>
        <h2>B) Ventajas de Reforestar</h2>
        <ul>
            <li>Mejora la calidad del aire</li>
            <li>Regula el clima</li>
            <li>Aumenta la disponibilidad de agua</li>
        </ul>
    </section>

    <section>
        <h2>C) Objetivos de la Reforestación</h2>
        <ul>
            <li>Mejorar el medio ambiente</li>
            <li>Proteger la biodiversidad</li>
            <li>Proteger el suelo</li>
        </ul>
    </section>

    <section>
        <h2>D) Tipos de Reforestación</h2>
        <ul>
            <li>Reforestación Rural</li>
            <li>Reforestación Urbana</li>
        </ul>
    </section>

    <section>
        <h2>E) Pasos en el Proceso de Reforestación</h2>
        <ol>
            <li>Conocer el terreno</li>
            <li>Definir la especie</li>
            <li>Desmalezar</li>

            <li>Elegir la época adecuada</li>

    <section>
        <h2>G) Especies para Reforestar en Querétaro</h2>
        <div class="especie">
            <strong>Encino (Quercus spp.)</strong><br>
            Árbol de copa densa que puede alcanzar 10-30 metros. Se planta durante la temporada de lluvias.<br>
            <em>Cuidados:</em> Riego constante inicial, suelos bien drenados, evitar fertilizantes químicos.
        </div>
        <div class="especie">
            <strong>Huizache (Vachellia farnesiana)</strong><br>
            Arbusto espinoso de 3-8 metros, resistente a la sequía. Se planta a finales de la temporada de lluvias.<br>
            <em>Cuidados:</em> Suelos áridos, riego ocasional, resistente a plagas.
        </div>
    </section>

    <section>
        <h2>H) Qué hacer antes y después de Reforestar</h2>
        <h3>Antes:</h3>
        <ul>
            <li>Conocer el terreno</li>
            <li>Elegir especies nativas</li>
            <li>Eliminar maleza</li>
            <li>Elegir la época adecuada</li>
        </ul>
        <h3>Después:</h3>
        <ul>
            <li>Proteger los árboles</li>
            <li>Prevenir plagas</li>
            <li>Evitar fogatas</li>
            <li>Hacer brechas cortafuego</li>
            <li>Riego adicional</li>
        </ul>
    </section>
